Debian: The Rock Stable Linux Distro.

Jun 5, 2026 · 5 min read


What is it?

Debian is a community maintained linux distro. Debian was founded in 1993. From that point until now, this distro is well known for it's rock-solid stability and solid sytem updates.

Many of our favourite and daily driving distros are based on debian. For example,

  1. Ubuntu
  2. LMDE (Linux Mint Debian Edition)
  3. MXLinux
  4. PopOS
  5. Kali Linux

Why I Prefer Debian?

  • No flashy updates, stays constant for too long.
  • Default repo contains only truly free and open source apps.
  • No flatpak or snap by default.
  • Simple and easier even for beginners.
  • It is the true foundation of many linux distros.
  • It have the huge number of packages in its repository.
  • Every single service who have a linux application provides .deb packages by default.
  • Less bloat compared to other main stream distros.
  • Users have no control over the installation like arch provides, but still most configs are decently configurable.
  • Default DE is Gnome.

What are the Downsides?

  • Older version of softwares.
  • Because of debians default DE is Gnome, users have to depend on extensions to add more functionality. This is inconvinent when compared to KDE like DE.
  • On a new major update most extensions can be broken but the fixes will be out faster (within a week).
  • Installing proprietary nvidia drivers are hard.
  • If a service not providing a .deb package or debian repo, users have to heavily depend upon flatpaks for that software.
  • Flatpaks are awesome but it have it's own sandboxed area, which leads it to take a decent amount of storage even for a smaller app just because the dependency it needs.
  • Users have to manually enable the non-free repo.
  • Default file system is ext4, which takes 5% of reserved space and users can't utilize the full system storage they have
    (Reason: if in future the complete storage has been consumed, this reserved space will be used to manage the system without lag or crash. In personal computers or laptops, which a user constantly checks storage space, this is an unnecessary feature)

Supported Desktop Environments

Debian supports many desktop environments and all are stable enough to use as daily drivers. Supported DE's are,

  • GNOME (Default)
  • KDE Plasma
  • Cinnamon
  • xfce
  • MATE
  • LXDE
  • LXQt

How to Actually Install Debian?

Installing Debian is easier. All you need is a USB drive or cd then an ISO file from their website. This guide is for netinstall.


Step-by-Step Installation Guide

  1. Download and flash the netinstall ISO file from Debian official website.
  2. Reboot into Boot menu (by pressing F12) and choose your installation medium.
  3. Choose 'Graphical install'
  4. Select Launguage
  5. Select location
  6. Configure the keyboard
  7. Configure the network
  8. Give your computer a name (Hostname)
  9. Setup a Root password (if you prefer using Sudo instead, skip this)
  10. New user full name
  11. Choose a username and password for new user (you have to remember it as it is required to login to system)
  12. Partiton disks (choose what you prefer, i usually choose 'use entire disk') and choose the disk to partition (Careful: never choose your live USB or CD).
  13. Partitioning scheme: All files in one partition (Recommended for new users)
  14. Finish partitioning and write changes to disk (default settings are ok for most users)
  15. Write changes to disk ? -> yes
  16. After some time it will ask you to choose a mirror. Country: your own (usually faster) -> debian archive mirror: deb.debian.org -> HTTP Proxy: None
  17. Popularity-contest: no
  18. Software selection (choose your DE and utilities): GNOME (Default and Stable)

That's it! Just wait until everything get downloaded and reboot to enjoy your new debian distro.
