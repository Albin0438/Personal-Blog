What is Debian?


Debian is a Linux distribution. It is well known for its rock solid stability. Each release supports about 3 to 5 years without major changes. It stays as it is and proven never break and interrupt user experience like other distribution.

Debian have its own package manager called apt with more than 60000+ packages. The package manager holds stable versions of many widely used software. When compared to the actual release, Debian packages are 3 to 4 versions behind but ensures its stability.

Note: Execute the commands with sudo if you are not a root user. If you are, avoid using sudo.

System Update

sudo apt update // search for updates
sudo apt upgrade // done existing updates

System Cleaning

sudo apt autoclean
automatically cleanup unused dependencies
sudo apt purge libreoffice*
completely remove everything with the keyword 'libreoffice'

System Storage

df
disk free in bytes
df -h
disk free in mega-bytes or giga-bytes
du /var/log
disk usage in a specific directory or sub-directory in bytes
du -h /var/log
disk usage in a specific directory or sub-directory in mega-bytes or giga-bytes
du -sh /var/log
a total summary about disk usage in mega-bytes or giga-bytes
du -h /var | sort -hr | head 20
top 20 storage consuming programs listed in reverse order (biggest ithem first) in mega-bytes or giga-bytes
du -h --max-depth=1
runs a deep analysis and calculate total size of current directory and its immediate subdirectories only
find / -type f -size +100M
scans the entire system and find items that exceeds size 100 mega-bytes
find . -type f -size +100M
scans currect directory and find items that exceeds size 100 mega-bytes
